イベント参加の心得

SNBコミュニティ主催イベント 共通ルール

制定日：2026年7月21日／運営：SNBコミュニティ

本心得は、SNBコミュニティが主催する撮影交流会、ユニフォーム交流会その他のイベントに、すべての参加者が安心して参加できる環境を維持するためのルールです。参加の申込みをもって、本心得および会場となる Studio Nagoya Base の利用規約に同意したものとみなします。

1. 参加資格

  1. 参加者は18歳以上に限ります。高校生は参加できません。
  2. 初参加の方には、年齢、簡単な自己紹介、参加目的その他運営が必要と判断する事項の確認をお願いする場合があります。
  3. 虚偽の申告、なりすまし、他の参加者を不安にさせる目的での参加は認めません。
  4. 運営は、安全な開催が難しいと判断した場合、参加をお断りすることがあります。判断理由については、必ずしも開示しません。

2. 基本的な参加姿勢

  1. 他の参加者の年齢、容姿、体型、性別、性的指向、経験、撮影技術、衣装その他の個人的な要素を尊重してください。
  2. 相手が不快に感じる発言、からかい、過度な品評、身体への不用意な接触は禁止します。
  3. 衣装や撮影への参加を強要してはいけません。
  4. 他の参加者が断った場合は、理由を求めたり、繰り返し依頼したりせず、その意思を尊重してください。
  5. イベントの趣旨から著しく外れる行為や、進行を妨げる行為は禁止します。

3. 撮影に関するルール

  1. 他の参加者を撮影する場合は、撮影前に本人の同意を得てください。
  2. 集合写真、オフショット、動画、ライブ配信についても同様です。
  3. 着替え中、休憩中、本人が撮影を希望していない場面での撮影は禁止します。
  4. 撮影への同意は、その写真の公開や第三者への提供まで許可するものではありません。
  5. 撮影した写真または動画をSNS、ウェブサイトその他の公開媒体へ掲載する場合は、写っている本人全員の事前確認を得てください。
  6. 一度掲載の同意を得た場合でも、本人から削除の申出があったときは、合理的な範囲で速やかに対応してください。
  7. 他の参加者が写り込んだ写真を、本人が特定できる状態のまま無断で公開してはいけません。
  8. イベント中のライブ配信、音声配信およびリアルタイムでの位置情報の公開は、運営の事前承認がある場合を除き禁止します。

4. 連絡先交換およびイベント後の交流

  1. SNSアカウントや連絡先の交換は、当事者双方が希望する場合に限ります。
  2. 連絡先の交換、フォロー、DMへの返信、個人的な交際、撮影への参加等を強要してはいけません。
  3. 断られた後の繰り返しの連絡、執拗な勧誘、待ち伏せ、監視に類する行為は禁止します。
  4. イベントで知り得た氏名、SNSアカウント、連絡先、職業その他の個人情報を、本人の許可なく第三者へ伝えてはいけません。
  5. イベント後の参加者間の交流は各自の責任で行ってください。ただし、イベントをきっかけとした迷惑行為について運営に相談があった場合は、今後の参加をお断りすることがあります。

5. 禁止する勧誘・営業行為

次の行為を禁止します。

  1. 宗教、政治団体、ネットワークビジネス、投資、商品またはサービスへの勧誘
  2. 運営の許可を得ていない有償撮影、モデル募集、イベント、店舗その他の営業活動
  3. 他の参加者から金銭を集める行為
  4. 他の参加者を別の集会、撮影、サービス等へ執拗に誘う行為
  5. その他、イベントを利用した営利目的または勧誘目的の行為

参加者同士が合意した個人的な撮影等を一律に禁止するものではありませんが、イベント中の無断営業は認めません。

6. 衣装および持ち物

  1. 各イベントで指定された服装、衣装または持ち物の条件に従ってください。
  2. 他者の権利を侵害する衣装、小道具、ロゴ、写真等を持ち込まないでください。
  3. 危険物、違法な物品、火気、周囲を負傷させるおそれのある小道具は持ち込めません。
  4. 貴重品および持ち物は各自で管理してください。

7. 安全および体調管理

  1. 体調が悪い場合、感染症が疑われる場合、または飲酒等により正常な参加が難しい場合は、参加を控えてください。
  2. イベント中に体調不良や危険を感じた場合は、速やかに運営へ申し出てください。
  3. 運営から安全確保、撮影中止、休憩、移動その他の指示があった場合は従ってください。
  4. 故意または過失により会場、機材、衣装その他の物品を破損した場合は、実費をご負担いただくことがあります。

8. 運営による記録撮影

  1. イベントの記録として、運営が写真または動画を撮影する場合があります。
  2. 運営による撮影物をSNS、ウェブサイト、告知物または作例として公開する場合は、公開対象となる参加者の意思を確認します。
  3. 撮影を希望しない方、または公開を希望しない方は、申込み時またはイベント開始前に運営へお知らせください。
  4. 撮影への参加と、運営による公開への同意は別に扱います。イベントへの参加を理由として、顔や姿の公開を一律に求めることはありません。

9. 違反時の対応

  1. 本心得に違反した場合、運営は注意、撮影の中止、写真の削除要請、退出、以後の参加禁止その他必要な対応を行うことがあります。
  2. 重大な違反、迷惑行為、虚偽申告または安全を脅かす行為については、事前の注意なく退出を求める場合があります。
  3. 違法行為または生命・身体に危険が及ぶ行為があった場合は、警察、救急機関その他の関係機関へ連絡する場合があります。
  4. 参加者の規約違反または自己都合により退出となった場合、参加費は原則として返金しません。ただし、運営が相当と認める場合、または法令上返金が必要となる場合はこの限りではありません。

10. 参加者間のトラブル

参加者間の個人的なやり取りや、イベント終了後の関係について、運営は当事者として責任を負いません。

ただし、イベント内での迷惑行為、本心得への違反、または今後の開催に影響する問題について相談を受けた場合は、事情を確認し、参加制限その他の必要な対応を行うことがあります。

11. 心得の改定

運営は、安全な開催および運営上の必要に応じて、本心得を改定することがあります。

改定後の内容は、ウェブサイトへの掲載その他運営が適切と判断する方法で告知し、告知した効力発生日から適用します。
